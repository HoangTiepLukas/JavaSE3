## Java SE III
**This project loads game data from games.csv file and generates 3 output files**
- game_renres.csv
- game_publishers.csv
- simulator_games.csv

**The application uses only two classes:**
- Game – data model
- Main – contains all logic for loading, parsing, filtering, sorting, and exporting data

**Technologies used**
- Java
- Maven
- Collections(List, Map, Set...)
- Basic file I/O: BufferedReader, BufferedWriter, Files

### Running the Program
Using IntelliJ IDEA
Open the project.

- Open Main.java.

- Click Run.

### Testing
- JUNIT 5 for testing
